What's my pick-of-the-week or tip-of-the-week?

My pick of the week this week actually comes from the Extreme Programming textbook, because I feel it will be good to keep in mind for the upcoming project. We will probably be required to connect to a database as part of this project, and we will most definitely have to write tests, so the following tip (taken from chapter 13 of the Extreme Programming Installed book) helps answer the question: How do you efficiently test when your program requires database connection? The answer: Test functionality by taking input from pre-created files. This keeps your tests fast and still ensures that you're doing things properly. As for the database connection itself, have a single test to make sure that your code connects to and reads from it properly, just to be safe.
